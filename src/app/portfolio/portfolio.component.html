Night City Club

Angular | JavaScript | Firebase | HTML | CSS

Welcome to Night City Club, a multiplayer game with discussions about the stories you've experienced within the Cyberpunk 2077 world while playing with your friends online. With the power of Firebase, every game state is stored, allowing you to pick up right where you left off using a unique game ID.
